## 🎯 游戏操作说明

### 🐍 贪吃蛇
- **方向键** ⬆⬇⬅➡ 控制蛇的移动方向
- 吃到红色果实得分，蛇身变长
- 撞墙或撞到自己身体则游戏结束
- 填满整个 20×20 棋盘即为胜利

### 📦 推箱子
- **方向键** 或 **WASD** 控制角色移动
- 将所有木箱推到黄色星标位置即可过关
- 共 5 个关卡，难度递增
- 箱子只能推不能拉，注意不要推到死角

### 🚀 星际战机·打怪兽
- **鼠标移动** 或 **手指滑动** 控制战机位置
- 战机**自动开火**，无需手动射击
- 三种怪兽：红色（+10 分）、紫色（+20 分）、蓝色（+30 分，2 点血量）
- 战机拥有 3 点生命值，被击中后短暂无敌

### 🔨 打地鼠
- **鼠标点击** 或 **触屏点击** 冒头的地鼠
- 普通地鼠 +10 分、黄金地鼠（带皇冠）+30 分、炸弹 -10 分
- 60 秒倒计时，时间到即结束

### 🦖 小恐龙跑酷
- **空格** 或 **⬆** 跳跃，**⬇** 下蹲
- 躲避仙人掌（地面）和飞虫（空中）
- 速度随时间逐渐加快，最高分自动保存

### ⚫ 五子棋
- **鼠标点击** 棋盘交叉点落子
- 黑棋先行，双方轮流落子
- 先将五颗棋子连成一线（横/竖/斜）者获胜
- 支持**悔棋**和**重新开局**

### 🧩 俄罗斯方块
- **⬅➡** 左右移动，**⬆** 旋转，**⬇** 加速下落，**空格** 硬降到底
- 消除满行得分：1 行 100 分、2 行 300 分、3 行 500 分、4 行 800 分
- 方块堆到顶部则游戏结束

## 🏗️ 技术亮点

### 单文件架构
每个游戏均为独立的 HTML 文件，CSS 样式与 JavaScript 逻辑全部内联，无需任何外部资源，开箱即用。

### Canvas 高性能渲染
采用 `requestAnimationFrame` 驱动游戏主循环，实现 60FPS 流畅动画。通过 AABB（轴对齐包围盒）碰撞检测算法实现精准的物理交互。

### 全端输入适配
完美兼容键盘、鼠标与移动端触摸事件，针对移动端做防误触滚动处理，保障操作精准度与体验一致性。

### 动态难度系统
多款游戏内置动态难度调节：小恐龙跑酷速度随分数递增、打地鼠地鼠出现频率随机、飞机打怪兽怪兽生成间隔随分数缩短。
